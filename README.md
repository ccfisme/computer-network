以创造者的角度来看，究竟是如何把一个电脑的数据传给另一个电脑的呢？ </br>
2.1 </br>
物理层是研究如何在不同的传输媒体上实现比特流的传输，而不是指具体的传输媒体，实现比特流的方式为通过定义传输媒体的接口形状，电压范围，以及每个电压代表什么，以及各个不同线路之间的连接过程这些特性，来确保比特流能从一台电脑传到另一台电脑</br>
